# Customs Broker Index

**Find and compare U.S. customs brokers.** → **[customsbrokerindex.com](https://customsbrokerindex.com)**

Customs Broker Index is an SEO-first directory of ~2,500 U.S. customs brokers, built so importers and freight forwarders can find a vetted broker by **city, specialty, or CBP port** — and request quotes through a built-in lead marketplace.

## What it does

- 🔎 **Searchable directory** of ~2,500 broker listings, enriched with specialties, ports served, phone, and website.
- 🏙️ **Programmatic pages** across cities × specialties and 43+ CBP ports — each targeting a distinct, high-intent search query.
- 📨 **Lead marketplace** — importers submit a "get matched" inquiry; qualified leads are distributed to relevant brokers automatically.
- 📝 **Automated content** — daily articles with an IndexNow + Google Search Console optimization loop.

## Who it's for

Importers, exporters, and freight forwarders who need licensed customs brokerage — and brokers who want qualified inbound leads.

## Tech stack

Astro (static SSG) · Supabase (PostgreSQL) · Vercel · Tailwind CSS · programmatic SEO
